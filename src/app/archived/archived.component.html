Project History

My most recent project was to create an author identifier. This was acheived through a stylometry algorithm that analized the word usage in each chunk of the book, and formed a shape based on that. Initlly this was preformed on all of the known works of a particular author and concatenated them together to create an author shape. This shape was then stored to a database. After this process was done with several authors, it then had the background info nessessary to operate. Once a book of unknown author is found, it is run through the program to determine its shape. This shape is compared to all the author shapes and the closest one is returned. It also takes the length of the book into considertion compared to each authors average book length.
One of my other projects was a imitation of rainbow road created using OpenGL. This project was for a graphics class so the main focus was on the visuals and how they interacted. The loop was one of the more difficult portions of the project, as it required the animated car to flip upside down once it reached a certain part of the loop. This project seems simple enough, but through the specialized format of OpenGl it was difficult.
